# Project Portfolio

Highlights of major programs and initiatives delivered in the last 5 years.

## License Optimisation

> Q3 2023 – Q2 2025

- Migrated IT asset management from HPAM to SnipeIT.
- Implemented CyberArk MFA migration to PrivacyIDEA.
- Localised software licences for development teams.
- Optimised licence usage for Loxon and OpenText platforms.
- Achieved significant cost reduction and compliance improvement.

## Oracle License Optimisation

> Q1 2024 – Q2 2025

- Established a dedicated WebLogic and Oracle Database cluster.
- Negotiated Oracle ULA agreement and optimised contract terms.
- Partnered with Krungsri & MUFG to consolidate enterprise licensing.
- Achieved up to 90% savings compared to the official Oracle price list.


## Azure Cost Optimisation

> Q3 2024 – Q2 2025

- Decommissioned unused Azure resources and optimised cloud spend.
- Right-sized compute and storage capacities across multiple environments.
- Enhanced cost transparency through native Azure monitoring tools.

## Azure & O365 Tenant Migration

> Q1 2024 – Q2 2025

- Led migration to a separate Microsoft tenant following divestiture.
- Migrated Entra ID (Azure AD) for ~2,800 users.
- Transitioned locally developed Azure applications to the new tenant.
- Ensured business continuity and security compliance during migration.

## Azure Cloud Adoption

> Q1 2021 – Q4 2023

- Drove adoption of Platform as a Service (PaaS) and Kubernetes (AKS).
- Managed cloud spending of approx. €1.5M annually.
- Introduced cognitive services for advanced analytics and automation.
- Established cloud governance and cost management framework.

## Open API Gateway Migration

> Q2 2024 – Q1 2025

- Migrated enterprise API management to a new Kong Gateway platform.
- Streamlined API publishing, monitoring, and authentication processes.
- Improved scalability, performance, and developer experience.

## Monitoring Platform Upgrade

> Q3 2024 – Present

- Transitioned monitoring from Splunk to Grafana Loki.
- Unified observability across all applications and infrastructure.
- Rebuilt monitoring dashboards for improved alerting and visibility.

## Spring Cloud Microservices to Kubernetes

> Q2 2022 – Q4 2023

- Migrated key front-facing applications from Spring Cloud orchestration to Kubernetes (AKS).
- Improved scalability, resiliency, and deployment automation.
- Enabled flexible scaling for peak season operations.

## Big Data Platform Upgrade

> Q1 2025 – Present

- Prepared RFI/RFP for a next-generation Big Data platform.
- Addressed end-of-support Cloudera Hadoop infrastructure.
- Optimised underlying hardware and improved data processing efficiency.

## Performance Fine-Tuning for Peak Season

> Q3 2023 – Present

- Led end-to-end performance testing across critical applications.
- Optimised database indices, APIs, and hardware configurations.
- Ensured high availability and optimal throughput during peak load periods.

## SOAP to REST Migration

> Q1 2024 – Q1 2025

- Replaced legacy SOAP APIs with modern RESTful interfaces.
- Implemented API rate limits and streamlined technical user management.
- Improved integration speed, maintainability, and security posture.
